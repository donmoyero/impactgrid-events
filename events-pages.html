How events work

After creating an event you can upload photos to it from the Upload Photos section.

Guests visit the event link, enter the code if required, and browse watermarked previews.

They can click Find My Photos to use face recognition, then request a download.

You approve or reject download requests from the Download Requests section.
30-day expiry
Events and all their photos are automatically deleted after the expiry date. Make sure the owner downloads what they need before then.

How reviews work

Guests submit reviews directly from the event gallery page. They land here as Pending.

When you click ✓ Approve, the review is:
• Marked approved in the Events database
• Mirrored into the main site database
• Displayed automatically in the Event Reviews section on the homepage

Rejecting or deleting also removes it from the homepage if it was previously approved.
